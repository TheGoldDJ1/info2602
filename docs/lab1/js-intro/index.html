Task 7

Write some code to record the height, weight and name of a collection of people. Then create a function which receives this collection and calculates their average bmi.

Answer

//Create a constructor a functions which builds object for us
function createPerson(name, height, weight) {
  return { name: name, height: height, weight: weight };
}

function calcBMI(weight, height) {
  return weight / (height * height);
}

function avgBMI(people) {
  let sum = 0;
  for (let person of people) {
    //sum the bmi of each person
    sum += calcBMI(person.weight, person.height);
  }
  //calculate average
  return sum / people.length;
}

//create a collection of people
let people = [
  createPerson("Sally", 60, 2.5),
  createPerson("Ben", 81, 3),
  createPerson("Shelly", 50, 1.7)
];

console.log(avgBMI(people));

Error handling is another crucial concept in front-end programming. Web apps should be able to catch errors and handle them gracefully without crashing. This is done with try catch statements.

function divide(a, b) {
  //It's good practice for functions to throw errors for
  //edge cases where the program is anticipated to fail
  if (b === 0) throw "Division by zero error";
  return a / b;
}

try {
  //any code which throw an error here would be caught
  console.log(divide(4, 2));//executes
  console.log(divide(20, 0));//throws error
  console.log(divide(10, 4));//does not execute
} catch (error) {
  //handle the error
  console.log(error);
} finally {
  //this runs regardless if an error was thrown
  console.log("Finish");
}

document.querySelector() is the function we use to access html elements on the page. Specifying â€˜#myForm' would retrieve the element with the id â€˜myForm'. We store that element in a myform variable.

Next we want to write to the page. We shall implement a drawTable() function which would use all of the objects in the records array to write rows into the table.

index.js 

let records = [];

function drawTable() {
        let myTbody = document.querySelector('#myTbody');
        let html = '';

      //create a table row for every record
        for (let record of records) {
                html += `<tr>
                             <td>${record.firstname}</td>
                             <td>${record.age}</td>
                             <td>${record.pref}</td>
                         </tr>`;
        }
        myTbody.innerHTML = html;
}

function addRecord() {
        let myform = document.querySelector('#myForm');

        let newRecord = {
                firstname: myform.elements['firstname'].value,
                age: myform.elements['age'].value,
                pref: myform.elements['pref'].value
        };

        records.push(newRecord);
        drawTable();
}

The draw table function builds an html string, interpolating td tags with the values of the records.

We use the innerHTML property to write the html string to the webpage. We then change the addRecord() function to call drawTable() everytime the add button is clicked.
